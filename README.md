# Sistema de Ventas - Base de Datos SQL

# Descripcion
Proyecto personal de base de datos relacional desarrollado en MySQL Workbench, enfocado en la gestión de clientes, productos, categorías y ventas. Incluye consultas SQL de nivel básico, intermedio y avanzado utilizando JOIN, GROUP BY, DISTINCT, HAVING y subconsultas.

# Tecnologias usadas
- MySQL Workbench
- SQL
- GitHub
- Windows 11

# Estructura de la base de datos

## Tablas principales
- clientes
- productos
- categorias
- ventas
- detalle_ventas

# Relaciones implementadas
- Un cliente puede realizar muchas ventas.
- Una venta puede contener muchos productos.
- Un producto pertenece a una categoría.
- La tabla detalle_ventas gestiona la relación entre ventas y productos.

## Consultas SQL practicadas
- SELECT

- ORDER BY

- GROUP BY
- HAVING
- COUNT()

- AVG()
- DISTINCT
- Subconsultas
- IN / NOT IN

- Productos más vendidos
- Clientes con mayor gasto
